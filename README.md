# ✋ SCT_ML_04 - Hand Gesture Recognition

## 📌 Overview

This project was completed as part of the **SkillCraft Technology Machine Learning Internship**.

The objective of this project is to develop a **Hand Gesture Recognition Model** that accurately identifies different hand gestures from image data using **Convolutional Neural Networks (CNN)**.

---

## 🎯 Objective

Build a deep learning model capable of recognizing and classifying hand gestures to enable gesture-based human-computer interaction.

---

## 🛠️ Technologies Used

- Python
- Google Colab
- OpenCV
- TensorFlow / Keras
- NumPy
- Matplotlib
- Scikit-learn

---

## 📂 Dataset

**LeapGestRecog (Hand Gesture Dataset)**

The dataset contains multiple hand gesture classes captured from different users.

---

## ⚙️ Project Workflow

1. Import required libraries
2. Load the hand gesture dataset
3. Resize and preprocess images
4. Normalize image pixel values
5. Split data into training and testing sets
6. Build a CNN model
7. Train the model
8. Evaluate model performance
9. Save the trained model

---

## 📊 Results

The CNN model successfully learned to classify different hand gestures with high accuracy after training.
